# Fundamental_Physics_2
>  **基础物理学 2 小班辅导讲义**

## 文件树

该仓库文件树如下，**半期小班辅导讲义**以及相应的 **$\LaTeX$ 源代码**均开源在本仓库，供学习和参考：

```shell
.
├── LICENSE
├── README.md
├── resourse
│   ├── bib
│   ├── compile.bat
│   ├── figure			# 讲义中用到的图片 
│   ├── material.bcf
│   ├── material.run.xml
│   ├── qbook.bcf
│   ├── qbook.cfg
│   ├── qbook.cls
│   ├── qbook.pdf
│   ├── qbook.run.xml
│   ├── qbook.synctex.gz
│   ├── qbook.tex   # TeX 编译主入口
│   ├── qbook.xdv
│   └── tex
└── 基础物理学2半期小班辅导讲义——学生版.pdf
```
